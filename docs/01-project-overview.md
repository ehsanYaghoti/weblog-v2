# Weblog v2

## overview

weblog version 2 is a modern blogging platform that allow users to publish articles and podcasts while enabling registered users to interact through comments , posts , bookmarks and likes.

this project is designed as a production-quality portfolio application that follow modern frontend development practices and will be later extended with ASP.NET core backend.

---

## Project Goals

### Primary Goal

Build a production-ready application that demonstrates professional frontend engineering skills.

### Secondary Goals

- Learn ASP.NET Core
- Learn SQL database design
- Practice clean architecture
- Improve UI/UX skills
- Learn deployment and maintenance
- Improve frontend skills

---

## Target Users

### Guest

- Browse articles
- Search articles
- Read and listen podcasts
- Register
- Login

### Registered User

- Comment on articles
- Like comments
- Bookmark articles
- Create posts
- Edit profile

### Author

- Create articles
- Edit articles
- Publish podcasts
- Manage drafts

### Administrator

- Manage users
- Manage permissions
- Manage roles
- Manage users
- Manage articles
- Manage categories
- Manage posts
- Assign roles
- Moderate comments
- Manage reports

---

## Project Scope

This project will focus on building a high-quality web application rather than implementing every possible feature.

Version 1 will prioritize stability, maintainability, and user experience.

---

## Success Criteria

The project will be considered complete when:

- All core features are implemented.
- Responsive on mobile and desktop.
- Accessibility standards are followed.
- TypeScript is used throughout.
- Code is well organized.
- Documentation is complete.
- Application is deployed.

---

## Out of Scope (Version 1)

The following features are intentionally excluded:

- AI article generation
- Mobile application
- Real-time chat
- Video streaming
- Multi-language support
